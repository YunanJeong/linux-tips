# codex-cli (2026.02.23.)

- codex라는 이름으로 커서처럼 IDE도 제공하고, 클로드코드처럼 CLI도 제공함
- 20달러짜리 ChatGPT Plus 이상 구독시 사용가능
- API KEY 기반 토큰종량제로도 사용가능
- 기본정책이 대화내역 등을 수집하지 않음
- gemini-cli에 비해 전반적으로 반응속도가 빠르고 조금 더 똑똑한 느낌이 있으나, 다소 답답한 부분도 있음
  - 맥락보다는 지시문장 그대로 실행하는 경향이 있었는데, `지시를 상당히 디테일하게 해야해서 바이브하다기보다 자연어"코딩"에 가까운` 느낌이었음. 
  - 이는 모델/사용방법에 따라 다를 수 있고, 취향차이일 듯
- 격리수준이 높아서, 로컬에 이미 배포된 서버에도 접근불가
  - e.g. 로컬 쿠버네티스 서버 127.0.0.1:6443에 접근을 못해서 사용자로 하여금 직접 배포하기를 요구함
